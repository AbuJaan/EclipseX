## Features

- **Audio Editing**: With commands like `reverb`, `slowed`, `nightcore`, and more, Eclipse X allows users to edit and manipulate audio files directly on Discord.
- **Image Editing**: Upload and manipulate images with ease using the `removebg` and `imgur-upload` commands.
- **Fun and Utility Commands**: Includes a variety of commands for fun, such as `ping`, `info`, `help`, and `support`.
- **Advanced Slash Commands**: All commands are available as slash commands, providing an easy and intuitive way to interact with the bot.

---

## Commands

### **Audio Editing Commands**

- **`/convert-audio`**: Convert audio files to different formats.
- **`/reverb`**: Add a reverb effect to your audio.
- **`/slowed`**: Slow down the speed of your audio.
- **`/spedup`**: Speed up the audio playback.
- **`/extract-audio`**: Extract audio from a video file.
- **`/reverse-audio`**: Reverse the audio for a cool effect.
- **`/nightcore`**: Speed up and increase the pitch of the audio for a nightcore effect.
- **`/audio-edit`**: A general command for audio editing tasks.
- **`/audio-cut `**: Split the audio from the given time.
- **`/replace-audio`**: Replace or mute the audio of a video file.

### **Image Editing Commands**

- **`/removebg`**: Remove the background from an image.
- **`/imgur-upload`**: Upload images to Imgur and get a shareable link.

## **Video Editing Commands**
- **`/extensions`**: Get After Effects and Premier Pro Extensions.
- **`/scripts`**: Gets After Effects and Premier Pro Scripts.
- **`/random-presets`**: Gets After Effects and Premier Pro Random Presets.
- **`/topaz-settings`**: Gets Topaz Settings for Higher Quality.
- **`/shake-presets`**: Gets After Effects and Premier Pro Shakes Presets.

### **Utility Commands**

- **`/help`**: Display a list of all available commands.
- **`/info`**: Get information about the bot, such as its version and status.
- **`/ping`**: Check the bot's latency and server connection.
- **`/bug`**: Report a bug or issue with the bot.
- **`/support`**: Get a link to the support server.
- **`/stats`**: View usage and detailed statistics for Eclipse X.
- **`/contribute`**: Contribute to the development of Eclipse X

## **Downloaders Command**

- **`/tiktok`**: Download any videos from TikTok without watermark.
- **`/Pinterest`**: Download any video or Image from Pinterest.
- **`/X`**: Download any video from X (Twitter).

## **Miscellaneous Command**
- **`/google`**: Google out anything without leaving Discord.
- **`/youTube`**: Search for YouTube videos without leaving Discord.
- **`/search-anime`**: Search for any Anime by image.

## **Trending Commands**
- **`/anime-tags`**: Get trending AMV/Anime tags.
- **`/trending-anime`**: Shows the top trending anime.

## **Server Commands**
- **`/listboosters`**: Displays the list of server boosters.
- **`/servericon`**: Displays the server's icon with download links.
- **`/serverinfo`**: Displays detailed information about the server.

## How to Use Eclipse X

Eclipse X is hosted and maintained by the author. To interact with the bot, simply invite it to your server and use the available commands through Discord.

### **Inviting Eclipse X to Your Server**

You can invite **Eclipse X** to your server using the following link:
[Invite Eclipse X](https://discord.com/oauth2/authorize?client_id=1352439744063082596&permissions=549756333280&integration_type=0&scope=bot)

### **Using Eclipse X**

Once the bot is added to your server, you can use the slash commands directly within Discord. Simply type `/` and you'll see the full list of available commands!
